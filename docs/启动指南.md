# CarMate 项目启动指南

## 后端

打开终端，分别执行：

```powershell
# 终端 1：主 API（登录、车牌、告警、历史、统计）
cd E:\carMate_vision\backend
python -m app.main
```

```powershell
# 终端 2：交警手势推理（需要 GPU 推理时才开）
cd E:\carMate_vision\backend
python server.py
```

## 前端

打开第三个终端：

```powershell
cd E:\carMate_vision\frontend
npm run dev
```

## 访问地址

| 地址 | 内容 |
|---|---|
| http://localhost:5173 | 前端页面 |
| http://localhost:8001/docs | 主 API 接口文档 |
| http://localhost:8000/docs | 推理服务接口文档 |
