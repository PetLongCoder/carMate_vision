# CarMate 项目启动指南

## 后端

打开终端：

```powershell
cd backend
pip install -r requirements.txt   # 首次运行需安装依赖
python -m app.main                # 启动主 API 服务
```

首次启动会自动从 `.env.example` 创建 `.env`，无需手动配置。

## 前端

打开第二个终端：

```powershell
cd frontend
npm install       # 首次运行需安装依赖
npm run dev       # 启动 Vite 开发服务器
```

## 访问地址

| 地址 | 内容 |
|---|---|
| http://localhost:5173 | 前端页面 |
| http://localhost:8001/docs | API 接口文档 |
| http://localhost:8001/api/health | 健康检查 |
